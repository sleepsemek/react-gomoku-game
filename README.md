# Gomoku

Веб-версия классической игры Гомоку (пять в ряд) на доске 15x15. Игроки поочередно ставят камни своего цвета. Цель - первым выстроить **пять камней в ряд** по горизонтали, вертикали или диагонали.
Если доска полностью заполнена и ни один игрок не выстроил пять камней подряд, объявляется **ничья**.

---

## Инструкция

* Черные делают первый ход
* **Мышью:** кликните на пустую клетку, чтобы поставить камень
* **Клавиатура:**
    * Стрелки перемещают курсор по доске
    * `Enter` ставит камень на текущую клетку

* **Новая игра:** кнопка "Новая игра" сбрасывает доску и историю ходов
* **Отменить ход:** кнопка "Отменить ход" убирает последний поставленный камень и возвращает очередь предыдущему игроку
* **История ходов:** последние 10 ходов отображаются справа с координатами и цветом камня
### Завершение игры
* Модальное окно оповещает о победителе
* Кнопка "Новая игра" запускает игру заново
